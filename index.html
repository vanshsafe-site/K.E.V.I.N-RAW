⌬ HOW TO USE K.E.V.I.N

🎤 Speak Mode

  • Kevin speaks motivational messages directly to you
  • Tap the orb to activate, then say more for a new message
  • Say keep talking for continuous support
  • Say thanks or exit to stop

👂 Listen Mode

  • Kevin listens — vent freely without any judgment
  • Tap the orb to begin. Speak as long as you need
  • Say Kevin stop when you're done talking
  • Kevin will respond with a supportive message
  • Say exit to end the session

⚡ Consult Mode

  • Tell Kevin what you're experiencing — e.g. "I have anxiety"
  • Kevin identifies the condition and provides tailored guidance
  • Over 100 mental health conditions recognized
  • Say again to repeat the advice
  • Say exit to end

📊 Charts Mode

  • View mental health lifestyle flowcharts on screen
  • Click a chart name button or tap orb and say the name
  • Kevin reads the chart aloud as it builds on screen
  • Available: anxiety depression stress self-esteem crisis sleep

⚠ Important Notes

  • Allow microphone access when prompted by your browser
  • Works best on Chrome or Edge (Web Speech API required)
  • Kevin is a supportive AI, not a licensed medical professional
  • Tap ✕ button or say exit at any time to reset
  • If tap-to-speak doesn't work, just say your command clearly
